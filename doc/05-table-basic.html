Aufgabe 05

Einfache Tabelle

🎯 Lernziel

Du stellst eine Liste von Datenobjekten in einer HTML-Tabelle dar. Du lernst map() zur Iteration, bedingte Formatierung von Zeilen und den Umgang mit Props-Arrays.

🔧 Konzepte

<table> <thead> / <tbody> / <tfoot> map() Conditional Rendering Props: Order[] CSS-Klasse bedingt setzen

📋 Aufgabe

  1. 1. Erstelle ein Interface Order mit den Properties price (number), volume (number), isBuyOrder (boolean), location (string).
  2. 2. Erstelle OrderTable.tsx mit einem Props-Array orders: Order[].
  3. 3. Baue eine HTML-Tabelle: <thead> mit Spaltenköpfen, <tbody> mit map() über die Orders.
  4. 4. Formatiere Buy-Orders mit grünem Hintergrund (className="table-success"), Sell-Orders mit rotem (className="table-danger").
  5. 5. Zeige in der Fußzeile (<tfoot>) die Gesamtanzahl der Einträge an.
  6. 6. Wenn die Liste null oder leer ist, zeige eine Meldung "Keine Daten vorhanden" statt der Tabelle.

📋 Beispieldaten zum Kopieren

// Interface
interface Order {
    price: number;
    volume: number;
    isBuyOrder: boolean;
    location: string;
}

// Beispieldaten
const orders: Order[] = [
    { price: 45.20, volume: 1000, isBuyOrder: true,  location: "Jita" },
    { price: 47.80, volume: 500,  isBuyOrder: false, location: "Amarr" },
    { price: 44.90, volume: 2500, isBuyOrder: true,  location: "Dodixie" },
    { price: 48.10, volume: 750,  isBuyOrder: false, location: "Jita" },
    { price: 45.50, volume: 1200, isBuyOrder: true,  location: "Rens" },
    { price: 49.00, volume: 300,  isBuyOrder: false, location: "Hek" },
];
💡 Hinweise – Code-Schnipsel

Die wichtigsten Bausteine:

// Interface (TypeScript):
interface Order {
    price: number;
    volume: number;
    isBuyOrder: boolean;
    location: string;
}

// Tabelle mit map():
<table className="table table-striped">
    <thead>
        <tr>
            <th>Preis</th>
            <th>Volumen</th>
            <th>Typ</th>
            <th>Station</th>
        </tr>
    </thead>
    <tbody>
        {orders.map(order => (
            <tr className={order.isBuyOrder ? "table-success" : "table-danger"}
                key={order.location + order.price}>
                <td>{order.price.toFixed(2)}</td>
                ...
            </tr>
        ))}
    </tbody>
    <tfoot>
        <tr><td colSpan={4}>
            {orders.length} Einträge
        </td></tr>
    </tfoot>
</table>

💡 Bedingte Anzeige mit {orders.length === 0 && <p>Keine Daten</p>}.

🏁 Vollständige Lösung 
{/* OrderTable.tsx */}

interface Order {
    price: number;
    volume: number;
    isBuyOrder: boolean;
    location: string;
}

interface OrderTableProps {
    title?: string;
    orders: Order[];
}

function OrderTable({ title = "Marktorder", orders }: OrderTableProps) {

    if (!orders || orders.length === 0) {
        return <p className="text-muted text-center py-3">Keine Daten vorhanden</p>;
    }

    return (
        <div className="card shadow">
            <div className="card-header fw-bold">{title}</div>
            <div className="card-body">
                <div className="table-responsive">
                    <table className="table table-striped table-hover mb-0">
                        <thead className="table-dark">
                            <tr>
                                <th>Preis (ISK)</th>
                                <th>Volumen</th>
                                <th>Typ</th>
                                <th>Station</th>
                            </tr>
                        </thead>
                        <tbody>
                            {orders.map(order => (
                                <tr className={order.isBuyOrder
                                    ? "table-success"
                                    : "table-danger"}
                                    key={order.location + order.price}>
                                    <td className="text-end">{order.price.toFixed(2)}</td>
                                    <td className="text-end">{order.volume.toLocaleString()}</td>
                                    <td>{order.isBuyOrder ? "Ankauf" : "Verkauf"}</td>
                                    <td>{order.location}</td>
                                </tr>
                            ))}
                        </tbody>
                        <tfoot>
                            <tr className="fw-bold">
                                <td colSpan={4} className="text-end">
                                    {orders.length} Einträge gesamt
                                </td>
                            </tr>
                        </tfoot>
                    </table>
                </div>
            </div>
        </div>
    );
}

💡 Tipps

  • Verwende item.price.toFixed(2) für formatierte Zahlen.
  • Bedingte CSS-Klasse: className={item.isBuyOrder ? "table-success" : "table-danger"}.
  • Gib einen title-Prop für die Card-Überschrift, damit die Komponente wiederverwendbar ist.
  • Überlege: Soll die Tabelle responsiv sein? → <div className="table-responsive"> drumherum.
  • Jeder <tr> braucht einen key für Reacts Reconciliation.

✅ Akzeptanzkriterien

  • Tabelle zeigt alle übergebenen Orders mit Price, Volume, Type und Location an
  • Buy-Orders haben grünen, Sell-Orders roten Hintergrund
  • Fußzeile zeigt die Gesamtanzahl der Einträge
  • Bei leerer Liste erscheint "Keine Daten vorhanden"
  • Zahlen sind lesbar formatiert (Tausendertrennzeichen, 2 Dezimalstellen)

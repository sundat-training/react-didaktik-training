07

Live-Suche / Autocomplete

🎯 Ziel dieser Einführung

Du baust ein Textfeld, das bei jeder Tastatureingabe eine Dropdown-Liste mit passenden Vorschlägen anzeigt – wie eine Autocomplete-Suche.

🔍 Das Beispiel zum Ausprobieren

Tippe etwas ein (z.B. "veld" oder "scor"): Die Liste filtert live. Klick auf einen Vorschlag übernimmt ihn.

Warte auf Eingabe...
📖 Was passiert hier?

Bei jeder Tastatureingabe wird gefiltert:

// Bei jedem Tastendruck
input.addEventListener('input', function() {
    var query = this.value.toLowerCase();
    var matches = items.filter(function(item) {
        return item.toLowerCase().includes(query);
    });
    // Treffer im Dropdown anzeigen
    renderDropdown(matches);
});

// Klick auf Treffer:
dropdown.addEventListener('click', function(e) {
    if (e.target.tagName === 'BUTTON') {
        input.value = e.target.textContent;
        dropdown.classList.remove('visible');
        // Wert übernehmen / callback feuern
    }
});

Problem: In React filterst du nicht das DOM, sondern einen State. Das Dropdown wird per {query && filtered.length > 0 && (...)} ein- und ausgeblendet.

🧩 Übertragung nach React
  • Eingabe: onChange an query State
  • Gefilterte Liste: const filtered = allItems.filter(i => i.toLowerCase().includes(query.toLowerCase()))
  • Dropdown: position: absolute; z-index: 1000; max-height: 300px; overflow-y: auto
  • Sichtbarkeit: {query && filtered.length > 0 && (...)}
  • Auswahl: onClick auf Item → Callback-Prop

💡 Bevor du loslegst

  • Wie schließt du das Dropdown bei Klick außerhalb? (Tipp: useRef + useEffect mit Event Listener)
  • Wie verhinderst du, dass das Dropdown bei leerem Input angezeigt wird?

Aufgabe 07

Live-Suche / Autocomplete

🎯 Lernziel

Du baust ein Textfeld, das bei jeder Tastatureingabe eine Dropdown-Liste mit passenden Vorschlägen anzeigt. Du lernst state-basierte Filterung (case-insensitive) via useState + onChange, das Positionieren eines absoluten Dropdowns und die Interaktion zwischen Texteingabe und Auswahl.

🔧 Konzepte

useState onChange filter() position: absolute z-index / overflow-y Conditional Rendering

📋 Aufgabe

  1. 1. Erstelle eine Komponente LiveSearch.tsx mit einem <input type="text"> und einem Dropdown-<div> darunter.
  2. 2. Bei onChange filterst du ein Array von Items case-insensitive nach dem eingegebenen Text.
  3. 3. Zeige die gefilterten Ergebnisse als Liste von <li>- oder <button>-Elementen im Dropdown an.
  4. 4. Positioniere das Dropdown mit position: absolute; z-index: 1000; max-height: 300px; overflow-y: auto, direkt unter dem Input.
  5. 5. Klick auf ein Ergebnis setzt den Input-Text.
  6. 6. Das Dropdown schließt bei Klick außerhalb (Tipp: useRef + useEffect mit mousedown-Listener).
  7. 7. Erweitere das Datenmodell: Statt string[] übergib ein Array von Objekten, die sowohl displayName als auch value haben.

// Case-insensitive Filterung:
const handleInput = (e: React.ChangeEvent<HTMLInputElement>) => {
    setSearchText(e.target.value);
    const q = e.target.value.toLowerCase();
    setFiltered(items.filter(x =>
        x.displayName.toLowerCase().includes(q)
        || x.value.toLowerCase().includes(q)
    ));
    setShowDropdown(true);
};

// onMouseDown statt onClick, damit onBlur nicht dazwischenfunkt:
const selectItem = (item: SearchItem) => {
    setSearchText(item.displayName);
    setShowDropdown(false);
    onItemSelected?.(item.value);
};

🏁 Vollständige Lösung 
// SearchItem.ts
export interface SearchItem {
    displayName: string;
    value: string;
}
/* LiveSearch.tsx */
import { useState } from "react";
import type { SearchItem } from "./SearchItem";

interface Props {
    items: SearchItem[];
    onItemSelected?: (value: string) => void;
}

export function LiveSearch({ items, onItemSelected }: Props) {
    const [searchText, setSearchText] = useState("");
    const [filtered, setFiltered] = useState<SearchItem[]>([]);
    const [showDropdown, setShowDropdown] = useState(false);

    const handleInput = (e: React.ChangeEvent<HTMLInputElement>) => {
        const q = e.target.value;
        setSearchText(q);
        setFiltered(items.filter(x =>
            x.displayName.toLowerCase().includes(q.toLowerCase()) ||
            x.value.toLowerCase().includes(q.toLowerCase())
        ));
        setShowDropdown(true);
    };

    const handleBlur = () => {
        setTimeout(() => setShowDropdown(false), 200);
    };

    const handleFocus = () => {
        if (searchText) setShowDropdown(true);
    };

    const selectItem = (item: SearchItem) => {
        setSearchText(item.displayName);
        setShowDropdown(false);
        onItemSelected?.(item.value);
    };

    return (
        <div style={{ position: "relative" }}>
            <input type="text"
                onChange={handleInput}
                onBlur={handleBlur}
                onFocus={handleFocus}
                value={searchText}
                placeholder="Suche..." />

            {showDropdown && filtered.length > 0 && (
                <div style={{
                    position: "absolute", width: "100%",
                    zIndex: 1000, maxHeight: 300, overflowY: "auto",
                    background: "#fff", border: "1px solid #ccc",
                    borderRadius: 4
                }}>
                    {filtered.map(item => (
                        <button key={item.value}
                            type="button"
                            className="dropdown-item"
                            onMouseDown={() => selectItem(item)}>
                            {item.displayName}
                            <small>({item.value})</small>
                        </button>
                    ))}
                </div>
            )}

            {showDropdown && !filtered.length && searchText && (
                <div style={{
                    position: "absolute", width: "100%",
                    zIndex: 1000, padding: 8,
                    background: "#fff", border: "1px solid #ccc"
                }}>
                    Keine Ergebnisse
                </div>
            )}
        </div>
    );
}
/* LiveSearch.css */
.dropdown-item:hover {
    background-color: #f1f3f5;
}

💡 Tipps

  • Verwende toLowerCase().includes() für case-insensitive Filterung.
  • Der Dropdown-Container muss position: relative sein, damit das absolute Dropdown korrekt ausrichtet.
  • Das Schließen bei Klick außerhalb: Nutze useRef + useEffect mit mousedown-Event auf document.
  • Wenn die gefilterte Liste leer ist, zeige "Keine Ergebnisse" im Dropdown an.
  • Markiere den Suchtreffer im Dropdown fett (substring + <strong>).

✅ Akzeptanzkriterien

  • Dropdown erscheint während der Eingabe unter dem Textfeld
  • Dropdown filtert case-insensitive in Echtzeit
  • Dropdown schließt bei Auswahl eines Eintrags
  • Dropdown schließt bei Klick außerhalb des Eingabebereichs
  • Auswahl setzt den Wert im Textfeld und feuert Callback
  • Bei leerer Trefferliste erscheint "Keine Ergebnisse"

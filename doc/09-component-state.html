Aufgabe 09

Component State & Kommunikation

🎯 Lernziel

Du kombinierst alle bisherigen Komponenten zu einer funktionsfähigen Seite. Du lernst das "Controller"-Pattern: Eine Parent-Komponente hält den gesamten State (useState) und verteilt ihn per Props an Child-Komponenten. Änderungen in den Kindern werden per Callback-Props zurückgemeldet.

🔧 Konzepte

interface Props callback props Parent→Child Datenfluss Lifting State Up Record<K,V> Conditional Rendering

📋 Aufgabe

  1. 1. Erstelle eine Seite Calculator.tsx – sie ist der "Controller". Sie hält mit useState: selectedItem, quantity, fee, result, isLoading, errorMessage.
  2. 2. Baue die Komponenten aus den vorherigen Aufgaben in die Seite ein:
    • Ein gekaskadetes Select (Aufgabe 02) für Kategorie + Item
    • Ein QuantityInput (Aufgabe 03) für die Menge
    • Ein DualInput (Aufgabe 04) für die Gebühr
    • Eine Tabelle (Aufgabe 05/06) für Ergebnisse
  3. 3. Verdrahte die Callback-Props der Kinder mit Handler-Funktionen in Calculator.tsx.
  4. 4. Wenn Item, Menge und Gebühr gültig sind, berechne (oder rufe API auf) und setze result.
  5. 5. Zeige während der Berechnung einen Spinner (isLoading). Bei Fehler eine rote Meldung (errorMessage).
  6. 6. Der "Berechnen"-Button ist deaktiviert, solange nicht alle Eingaben gültig sind.
  7. 7. Sorge dafür, dass bei Änderung einer Eingabe das vorherige Ergebnis zurückgesetzt wird.

// Ergebnis zurücksetzen bei neuer Eingabe:
const handleItemSelected = (item: string) => {
  setSelectedItem(item);
  setResult(null); // Altes Ergebnis löschen
};

// Async Calculation mit Loading-State:
const calculateAsync = async () => {
  setIsLoading(true);
  setErrorMessage(null);
  try {
    const res = await someService.calculateAsync(...);
    setResult(res);
  } catch (ex) {
    setErrorMessage(ex instanceof Error ? ex.message : "Fehler");
  } finally {
    setIsLoading(false);
  }
};

🏁 Vollständige Lösung 
// CalculationResult.ts
interface CalculationResult {
  itemName: string;
  quantity: number;
  totalPrice: number;
  fee: number;
  netPrice: number;
  orders: Order[];
}
{/* Calculator.tsx */}
import { useState } from "react";

function Calculator() {
  const [selectedCategory, setSelectedCategory] = useState("");
  const [selectedItem, setSelectedItem] = useState("");
  const [currentItems, setCurrentItems] = useState<string[]>([]);
  const [quantity, setQuantity] = useState(0);
  const [fee, setFee] = useState(2.5);
  const [result, setResult] = useState<CalculationResult | null>(null);
  const [isLoading, setIsLoading] = useState(false);
  const [errorMessage, setErrorMessage] = useState<string | null>(null);

  const data: Record<string, string[]> = {
    "Ordinary": ["Veldspar", "Scordite", "Pyroxeres"],
    "Compressed": ["Compressed Veldspar", "Compressed Scordite"],
    "Moon": ["Bitumens", "Carnotite", "Cinnabar"],
    "Compressed Moon": ["Compressed Bitumens", "Compressed Carnotite"],
  };

  const canCalculate = !!selectedItem && quantity > 0 && fee >= 0;

  const handleCategoryChanged = (cat: string) => {
    setSelectedCategory(cat);
    setCurrentItems(data[cat] ?? []);
    setSelectedItem("");
    setResult(null);
  };

  const handleItemSelected = (item: string) => {
    setSelectedItem(item);
    setResult(null);
  };

  const handleQuantityChanged = (val: number) => {
    setQuantity(val);
    setResult(null);
  };

  const handleFeeChanged = (val: number) => {
    setFee(val);
    setResult(null);
  };

  const calculateAsync = async () => {
    if (!canCalculate) return;

    setIsLoading(true);
    setErrorMessage(null);
    setResult(null);

    try {
      {/* Simulierte Berechnung (hier später echter API-Call) */}
      await new Promise(r => setTimeout(r, 500));
      const total = quantity * 42.42;
      const net = total * (1 - fee / 100);
      setResult({
        itemName: selectedItem,
        quantity,
        totalPrice: total,
        fee,
        netPrice: net,
        orders: [],
      });
    } catch (ex) {
      setErrorMessage(ex instanceof Error ? ex.message : "Fehler");
    } finally {
      setIsLoading(false);
    }
  };

  return (
    <div>
      <h3 className="mb-4">Ore Calculator</h3>

      <div className="row">
        <div className="col-md-5">
          <div className="card shadow">
            <div className="card-body">
              <CategorySelect onCategoryChanged={handleCategoryChanged} />
              <ItemSelect items={currentItems}
                          disabled={!selectedCategory}
                          onItemSelected={handleItemSelected}
                          key={selectedCategory} />
              <QuantityInput onQuantityChanged={handleQuantityChanged}
                             disabled={!selectedItem} />
              <DualInput onValueChanged={handleFeeChanged} />

              <button className="btn btn-primary w-100 mt-3"
                      onClick={calculateAsync}
                      disabled={!canCalculate}>
                {isLoading ? "Berechne..." : "Berechnen"}
              </button>
            </div>
          </div>
        </div>

        <div className="col-md-7">
          {isLoading && (
            <div className="text-center py-5">
              <span className="spinner-border text-primary"
                    style={{ width: "3rem", height: "3rem" }}></span>
              <p className="mt-2 text-muted">Berechne Preise...</p>
            </div>
          )}
          {!isLoading && errorMessage && (
            <div className="alert alert-danger">
              <strong>Fehler:</strong> {errorMessage}
            </div>
          )}
          {!isLoading && !errorMessage && result && (
            <div className="card shadow">
              <div className="card-header fw-bold">
                Ergebnis für {result.itemName} (x {result.quantity})
              </div>
              <div className="card-body">
                <table className="table mb-0">
                  <tbody>
                    <tr><td>Gesamtpreis</td><td className="text-end">{result.totalPrice.toFixed(2)} ISK</td></tr>
                    <tr><td>Gebühr ({result.fee.toFixed(1)}%)</td><td className="text-end text-danger">-{(result.totalPrice * result.fee / 100).toFixed(2)} ISK</td></tr>
                    <tr className="fw-bold"><td>Netto</td><td className="text-end">{result.netPrice.toFixed(2)} ISK</td></tr>
                  </tbody>
                </table>
              </div>
            </div>
          )}
        </div>
      </div>
    </div>
  );
}

💡 Tipps

  • Die Controller-Seite ruft kein manuelles Re-Rendering auf (React macht das automatisch bei State-Änderungen).
  • Definiere ein Interface für das Ergebnis, das alle relevanten Felder bündelt.
  • Der disabled-Zustand des Berechnen-Buttons ergibt sich aus: Item != null && quantity > 0 && fee >= 0.
  • Nutze key auf Komponenten, um React bei State-Änderungen ein sauberes Re-Rendering zu ermöglichen.

✅ Akzeptanzkriterien

  • Seite enthält: Item-Auswahl, Mengen-Eingabe, Gebühren-Eingabe, Ergebnis-Tabelle
  • Alle Child-Komponenten werden per Props gesteuert
  • Alle Child-Änderungen werden per Callback-Prop an die Parent gemeldet
  • Berechnen-Button ist deaktiviert bei ungültigen Eingaben
  • Spinner wird während der Berechnung angezeigt
  • Fehler werden als rote Meldung dargestellt
  • Bei Änderung einer Eingabe wird das vorherige Ergebnis zurückgesetzt

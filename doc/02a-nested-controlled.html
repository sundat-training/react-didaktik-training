Aufgabe 02a

Gekaskadete Selects controlled

🎯 Lernziel

Du baust zwei <select>-Dropdowns, die voneinander abhängen: Die Auswahl im ersten Dropdown bestimmt, welche Optionen das zweite anzeigt. Du lernst, wie man useState, Props und Callback-Funktionen nutzt, um Daten zwischen Parent- und Child-Komponenten weiterzureichen.

🔧 Konzepte

interface Props useState onChange Parent→Child Datenfluss Record<K,V> Conditional Rendering key ChangeEvent

📋 Aufgabe

  1. 1. Erstelle zwei Komponenten: CategorySelect.tsx (Kategorie-Auswahl) und ItemSelect.tsx (Item-Auswahl).
  2. 2. CategorySelect gibt die gewählte Kategorie per Callback-Prop nach oben.
  3. 3. ItemSelect nimmt per Props eine Liste von Items entgegen und zeigt sie im Dropdown an.
  4. 4. Erstelle eine Parent-Seite, die beide Komponenten einbettet. Bei Kategorie-Wechsel wird die Item-Liste neu gesetzt.
  5. 5. Verwende ein Record<string, string[]> als Datenquelle: Jeder Kategorie sind genau die Items zugeordnet, die in ItemSelect erscheinen.
  6. 6. Sorge dafür, dass bei Kategorie-Wechsel die Item-Auswahl zurückgesetzt wird.

📋 Beispieldaten zum Kopieren

const oreData: Record<string, string[]> = {
  "Ordinary": ["Veldspar", "Scordite", "Pyroxeres", "Plagioclase"],
  "Compressed": ["Compressed Veldspar", "Compressed Scordite", "Compressed Pyroxeres"],
  "Moon": ["Bitumens", "Carnotite", "Cobaltite"],
  "Compressed Moon": ["Compressed Bitumens", "Compressed Carnotite"],
};

// key für sauberes Re-Rendering:

🏁 Vollständige Lösung 
{/* CategorySelect.tsx (controlled) */}

interface CategorySelectProps {
  onCategoryChanged?: (category: string) => void;
}

function CategorySelect({ onCategoryChanged }: CategorySelectProps) {
  const [selectedCategory, setSelectedCategory] = useState("");

  const categories = ["Ordinary", "Compressed", "Moon", "Compressed Moon"];

  const handleChange = (e: React.ChangeEvent<HTMLSelectElement>) => {
    const cat = e.target.value;
    setSelectedCategory(cat);
    if (cat && onCategoryChanged) onCategoryChanged(cat);
  };

  return (
    <div className="mb-3">
      <label className="form-label fw-bold">Kategorie</label>
      <select className="form-select"
              onChange={handleChange}
              value={selectedCategory}>
        <option value="" disabled>Bitte wählen...</option>
        {categories.map(cat => (
          <option key={cat} value={cat}>{cat}</option>
        ))}
      </select>
    </div>
  );
}
{/* ItemSelect.tsx (controlled) */}

interface ItemSelectProps {
  items: string[];
  disabled?: boolean;
  onItemSelected?: (item: string) => void;
}

function ItemSelect({ items, disabled, onItemSelected }: ItemSelectProps) {
  const [selectedItem, setSelectedItem] = useState("");

  const handleChange = (e: React.ChangeEvent<HTMLSelectElement>) => {
    const item = e.target.value;
    setSelectedItem(item);
    if (item && onItemSelected) onItemSelected(item);
  };

  return (
    <div className="mb-3">
      <label className="form-label fw-bold">Item</label>
      <select className="form-select"
              onChange={handleChange}
              value={selectedItem}
              disabled={disabled}>
        <option value="" disabled>Bitte wählen...</option>
        {items.map(item => (
          <option key={item} value={item}>{item}</option>
        ))}
      </select>
    </div>
  );
}

Parent-Seite: wie in Teil-Lösung, mit handleCategoryChanged und handleItemSelected vervollständigt.

💡 Tipps

  • Die Parent-Komponente ist der "Controller": Sie hält den State und gibt ihn per Props an die Kinder weiter.
  • Setze ItemSelect auf disabled, solange keine Kategorie gewählt ist – das signalisiert dem Nutzer den Abhängigkeitsstatus.
  • Für das Zurücksetzen: übergib ein leeres Array oder setze den selektierten Wert auf "".
  • Verwende key auf ItemSelect, um React zu signalisieren, dass es bei Kategorie-Wechsel neu initialisieren soll.

✅ Akzeptanzkriterien

  • Kategorie-Dropdown zeigt alle Kategorien an
  • Item-Dropdown zeigt nur Items der gewählten Kategorie
  • Bei Kategorie-Wechsel wird die Item-Auswahl zurückgesetzt
  • Item-Dropdown ist deaktiviert, solange keine Kategorie gewählt ist
  • Die Auswahl beider Dropdowns wird auf der Seite angezeigt

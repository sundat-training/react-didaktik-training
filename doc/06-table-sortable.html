Aufgabe 06

Sortierbare Tabelle mit Filter

🎯 Lernziel

Du erweiterst die Tabelle aus Aufgabe 05 um clientseitige Sortierung per Klick auf Spaltenköpfe und um einen Filter (z.B. Alle / Buy / Sell). Du lernst den Umgang mit Array.sort(), Array.filter() und der Sortierrichtung als Component-State.

🔧 Konzepte

Array.sort Array.filter keyof T Sortier-Status onClick auf <th> Unicode ▲▼ Conditional Rendering

📋 Aufgabe

  1. 1. Kopiere die Komponente aus Aufgabe 05 und erweitere sie zu SortableTable.tsx.
  2. 2. Definiere einen Sortier-Status: sortKey: keyof Order (z.B. "price", "volume") und sortAsc: boolean.
  3. 3. Mache die <th>-Elemente klickbar. Bei Klick: Wechsel der Sortierrichtung bei gleicher Spalte, sonst neue Spalte aufsteigend.
  4. 4. Zeige einen Sortier-Indikator an: " ▲" oder " ▼" neben der aktiven Spalte.
  5. 5. Implementiere eine Filter-Button-Gruppe oberhalb der Tabelle: Alle | Buy | Sell.
  6. 6. Wende Filter vor der Sortierung an. Die ursprüngliche Props-Liste bleibt unverändert.

    if (sortKey === key) {

// Kombinierte Filter + Sortier-Liste:

    .filter(o => filter === "all"
        || (filter === "buy" && o.isBuyOrder)
        || (filter === "sell" && !o.isBuyOrder))
    .sort((a, b) => {
        const aVal = a[sortKey];
        const bVal = b[sortKey];

                : (bVal as string).localeCompare(aVal);

🏁 Vollständige Lösung 
{/* SortableTable.tsx */}

import { useState } from "react";

interface Order {
    price: number;
    volume: number;
    isBuyOrder: boolean;
    location: string;
}

const initialOrders: Order[] = [
    { price: 45.20, volume: 1000, isBuyOrder: true,  location: "Jita" },
    { price: 47.80, volume: 500,  isBuyOrder: false, location: "Amarr" },
    { price: 44.90, volume: 2500, isBuyOrder: true,  location: "Dodixie" },
    { price: 48.10, volume: 750,  isBuyOrder: false, location: "Jita" },
    { price: 45.50, volume: 1200, isBuyOrder: true,  location: "Rens" },
    { price: 49.00, volume: 300,  isBuyOrder: false, location: "Hek" },
];

type FilterType = "all" | "buy" | "sell";

function SortableTable() {
    const [sortKey, setSortKey] = useState<keyof Order>("price");
    const [sortAsc, setSortAsc] = useState(true);
    const [filter, setFilter] = useState<FilterType>("all");

    const toggleSort = (key: keyof Order) => {
        if (sortKey === key) {
            setSortAsc(!sortAsc);
        } else {
            setSortKey(key);
            setSortAsc(true);
        }
    };

    const getSortIcon = (key: keyof Order) => {
        if (sortKey !== key) return "";
        return sortAsc ? " ▲" : " ▼";
    };

    const processed = [...initialOrders]
        .filter(o => {
            if (filter === "all") return true;
            return filter === "buy" ? o.isBuyOrder : !o.isBuyOrder;
        })
        .sort((a, b) => {
            const aVal = a[sortKey];
            const bVal = b[sortKey];
            if (typeof aVal === "string") {
                return sortAsc
                    ? aVal.localeCompare(bVal as string)
                    : (bVal as string).localeCompare(aVal);
            }
            return sortAsc
                ? (aVal as number) - (bVal as number)
                : (bVal as number) - (aVal as number);
        });

    return (
        <div className="card shadow">
            <div className="card-header d-flex justify-content-between align-items-center">
                <span className="fw-bold">Marktorder</span>
                <div className="btn-group btn-group-sm">
                    <button className={filter === "all" ? "btn btn-primary" : "btn btn-outline-primary"}
                            onClick={() => setFilter("all")}>Alle</button>
                    <button className={filter === "buy" ? "btn btn-success" : "btn btn-outline-success"}
                            onClick={() => setFilter("buy")}>Ankauf</button>
                    <button className={filter === "sell" ? "btn btn-danger" : "btn btn-outline-danger"}
                            onClick={() => setFilter("sell")}>Verkauf</button>
                </div>
            </div>

            <div className="card-body">
                {processed.length === 0 ? (
                    <p className="text-muted text-center py-2">Keine passenden Einträge</p>
                ) : (
                    <div className="table-responsive">
                        <table className="table table-hover mb-0">
                            <thead className="table-dark">
                                <tr>
                                    <th onClick={() => toggleSort("price")} role="button">
                                        Preis{getSortIcon("price")}</th>
                                    <th onClick={() => toggleSort("volume")} role="button">
                                        Volumen{getSortIcon("volume")}</th>
                                    <th onClick={() => toggleSort("type")} role="button">
                                        Typ{getSortIcon("type")}</th>
                                    <th onClick={() => toggleSort("location")} role="button">
                                        Station{getSortIcon("location")}</th>
                                </tr>
                            </thead>
                            <tbody>
                                {processed.map(order => (
                                    <tr className={order.isBuyOrder ? "table-success" : "table-danger"}
                                        key={order.location + order.price}>
                                        <td className="text-end">{order.price.toFixed(2)}</td>
                                        <td className="text-end">{order.volume.toLocaleString()}</td>
                                        <td>{order.isBuyOrder ? "Ankauf" : "Verkauf"}</td>
                                        <td>{order.location}</td>
                                    </tr>
                                ))}
                            </tbody>
                        </table>
                    </div>
                )}
            </div>
        </div>
    );
}

💡 Tipps

  • Verwende keyof Order für typsichere Spaltennamen.
  • Sortiere mit [...orders].sort() – erst kopieren, dann sortieren, um das Original nicht zu mutieren.
  • Der aktive Filter-Button sollte optisch hervorgehoben sein (z.B. btn-primary vs btn-outline-primary).
  • Unicode-Dreiecke: " ▲" und " ▼".
  • Filter zuerst, dann sortieren – die Chain lautet filter().sort().

✅ Akzeptanzkriterien

  • Klick auf Spaltenkopf sortiert die Tabelle (auf-/absteigend wechselnd)
  • Aktive Sortierspalte zeigt ▲ oder ▼ an
  • Filter-Buttons (Alle/Buy/Sell) filtern die Tabelle korrekt
  • Filter und Sortierung kombinieren: erst filtern, dann sortieren
  • Die ursprüngliche Props-Liste wird nicht verändert
  • Bei leerem Filterergebnis erscheint eine Hinweismeldung

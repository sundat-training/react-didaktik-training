📖 Wie funktioniert das?

CSS-Variablen (Custom Properties) machen den Theme-Wechsel möglich:

// Hell (Standard)
:root {
    --bg-card: #f8f9fa;
    --border: #dee2e6;
    --text: #212529;
    --accent: #4361ee;
}

// Dunkel via Attribut-Selektor
[data-theme="dark"] {
    --bg-card: #25262b;
    --border: #373a40;
    --text: #e9ecef;
    --accent: #5c7cfa;
}

Ein JS-Click toggled data-theme aufs <html>-Element. Alle Elemente, die CSS-Variablen nutzen, wechseln sofort mit.

In React: useEffect + DOM-Manipulation (document.documentElement.setAttribute) setzt data-theme und speichert die Präferenz in localStorage.

🧩 CSS Modules in React
  • CSS Modules: Button.module.css – Styles gelten nur innerhalb dieser Komponente
  • :global: Überschreibt die Modul-Grenze, um globale Styles zu setzen
  • CSS-Variablen: Global in :root definiert, in Komponenten referenziert
  • Dark Mode: data-theme="dark" auf <html> + andere Variablen-Werte
  • localStorage: Präferenz speichern, damit sie beim nächsten Besuch erhalten bleibt

Aufgabe 07b

Live-Suche / Autocomplete als Controlled Component

🎯 Lernziel

Du löst dieselbe Aufgabe wie in 07, aber als wiederverwendbare Komponente mit Props. Der Input wird von außen über value + onChange gesteuert (Controlled Component). Die Filterung läuft automatisch über einen useEffect-Hook.

🔧 Konzepte

useState Props onChange Controlled Input Callback-Prop case-insensitive Filter

📋 Aufgabe

  1. 1. Erstelle SearchSelect.tsx mit Props items: SearchItem[] und onItemSelected?: (value: string) => void.
  2. 2. Der Input ist ein Controlled Component: value={searchText} + onChange={e => setSearchText(e.target.value)}.
  3. 3. Ein useEffect filtert bei jeder Änderung von searchText die Items case-insensitive.
  4. 4. Zeige die Treffer als <button>-Liste im absolut positionierten Dropdown.
  5. 5. Klick auf ein Ergebnis setzt searchText und feuert onItemSelected.

📌 Der Unterschied zu 07

In 07 filterst du manuell im onChange-Handler. Hier übernimmt useEffect die Filterung automatisch, sobald sich searchText ändert – getrennte Verantwortlichkeiten.

🏁 Vollständige Lösung 
/* SearchSelect.tsx */

import { useState, useEffect } from "react";
import type { SearchItem } from "./SearchItem";

interface Props {
    items: SearchItem[];
    onItemSelected?: (value: string) => void;
}

export function SearchSelect({ items, onItemSelected }: Props) {
    const [searchText, setSearchText] = useState("");
    const [filtered, setFiltered] = useState<SearchItem[]>([]);

    useEffect(() => {
        const q = searchText.toLowerCase();
        setFiltered(
            searchText
                ? items.filter(i =>
                    i.displayName.toLowerCase().includes(q) ||
                    i.value.toLowerCase().includes(q))
                : []
        );
    }, [searchText, items]);

    const selectItem = (item: SearchItem) => {
        setSearchText(item.displayName);
        setFiltered([]);
        onItemSelected?.(item.value);
    };

    return (
        <div style={{ position: "relative" }}>
            <input type="text"
                value={searchText}
                onChange={e => setSearchText(e.target.value)}
                placeholder="Item suchen..." />

            {filtered.length > 0 && (
                <div className="dropdown-menu show" style={{
                    position: "absolute", width: "100%",
                    zIndex: 1000, maxHeight: 300, overflowY: "auto"
                }}>
                    {filtered.map(item => (
                        <button className="dropdown-item"
                            key={item.value}
                            onClick={() => selectItem(item)}
                            type="button">
                            {item.displayName}
                        </button>
                    ))}
                </div>
            )}
        </div>
    );
}

💡 Tipps

  • Ein Controlled Component bekommt seinen Wert über value und aktualisiert ihn über onChange – React steuert den State.
  • Das Dropdown schließt sich, wenn filtered leer ist oder ein Item ausgewählt wurde.
  • Für Klick-außerhalb-schließen: Verwende useRef + useEffect mit mousedown-Event auf document.

✅ Akzeptanzkriterien

  • Dropdown zeigt gefilterte Vorschläge während der Eingabe
  • Filter ist case-insensitive
  • Klick auf einen Vorschlag setzt den Input-Text und feuert Callback
  • Es wird value + onChange (Controlled Component) und useEffect für die Filterung verwendet

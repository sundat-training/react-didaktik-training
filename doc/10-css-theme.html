Aufgabe 10

CSS-Techniken & Theme

🎯 Lernziel

Du optimierst das visuelle Erscheinungsbild der Anwendung. Du lernst CSS Modules (CSS Isolation), den :global-Selector, die Implementierung eines Dark/Light-Mode-Wechsels mit useEffect und responsive Breakpoints.

🔧 Konzepte

CSS Modules (.module.css) :global data-theme CSS-Variablen (custom properties) @media (prefers-color-scheme) @media (max-width) useEffect localStorage

📋 Aufgabe

  1. 1. Erstelle für jede deiner Komponenten eine CSS Module-Datei (Komponente.module.css). Verschiebe komponenten-spezifische Styles aus globalem CSS dorthin.
  2. 2. Nutze :global in einer Parent-Komponente, um das Styling einer Child-Komponente zu beeinflussen (z.B. alle Tabellen in einer Card einheitlich formatieren).
  3. 3. Implementiere einen Dark/Light-Mode-Button:
    • Ein Button togglet per useEffect das data-theme-Attribut auf <html>.
    • Die Präferenz wird in localStorage gespeichert und beim Laden angewendet.
  4. 4. Ergänze benutzerdefinierte CSS-Variablen für Dark Mode (z.B. andere Sidebar-Hintergrundfarbe, andere Card-Schatten).
  5. 5. Mache die Sidebar responsiv: Unter 640px Breite wird sie zu einem Hamburger-Menü, das sich über den Content legt.
  6. 6. Bonus: Füge prefers-color-scheme als Standard hinzu, damit der Dark Mode automatisch der Betriebssystem-Einstellung folgt.
💡 Hinweise – Code-Schnipsel

Die wichtigsten Bausteine:

    color: var(--text-muted);

    border-radius: 6px;
    transition: all 0.15s;
}

:global .nav-link.active {
    color: var(--accent);
    background: var(--accent-light);
}

// CSS-Variablen für Dark Mode:
:root {
    --sidebar-bg: #f8f9fa;
    --card-shadow: 0 1px 3px rgba(0,0,0,0.12);
}

[data-theme="dark"] {
    --sidebar-bg: #212529;
    --card-shadow: 0 1px 3px rgba(0,0,0,0.4);
}

// React-Komponente mit useEffect für Theme-Toggle:
function ThemeToggle() {
  const [dark, setDark] = useState(
    window.matchMedia("(prefers-color-scheme: dark)").matches
  );

  useEffect(() => {
    document.documentElement.setAttribute(
      "data-theme", dark ? "dark" : "light"
    );
    localStorage.setItem("theme", dark ? "dark" : "light");
  }, [dark]);

  useEffect(() => {
    const saved = localStorage.getItem("theme");
    if (saved) {
      document.documentElement.setAttribute("data-theme", saved);
      setDark(saved === "dark");
    }
  }, []);

  return (
    <button onClick={() => setDark(!dark)}>
      {dark ? "☀️ Helles Theme" : "🌙 Dunkles Theme"}
    </button>
  );
}

🏁 Vollständige Lösung 
// utils/theme.ts
export function toggleTheme(): boolean {
  const html = document.documentElement;
  const current = html.getAttribute("data-theme") || "light";
  const next = current === "dark" ? "light" : "dark";
  html.setAttribute("data-theme", next);
  localStorage.setItem("theme", next);
  return next === "dark";
}

export function applyTheme(): boolean {
  const saved = localStorage.getItem("theme");
  if (saved) {
    document.documentElement.setAttribute("data-theme", saved);
    return saved === "dark";
  }
  const prefersDark = window.matchMedia("(prefers-color-scheme: dark)").matches;
  if (prefersDark) {
    document.documentElement.setAttribute("data-theme", "dark");
  }
  return prefersDark;
}
// NavMenu.tsx (Auszug)
function NavMenu() {
  const [isDark, setIsDark] = useState(false);
  const [navOpen, setNavOpen] = useState(false);

  useEffect(() => {
    setIsDark(applyTheme());
  }, []);

  useEffect(() => {
    document.documentElement.setAttribute(
      "data-theme", isDark ? "dark" : "light"
    );
    localStorage.setItem("theme", isDark ? "dark" : "light");
  }, [isDark]);

  return (
    <nav className="flex-column">
      <div className="d-md-none text-end p-2">
        <button className="btn btn-sm btn-outline-secondary"
                onClick={() => setNavOpen(!navOpen)}>
          ☰
        </button>
      </div>

      <div className={navOpen ? "d-block" : "d-none d-md-block"}>
        <a className="nav-link" href="/">
          🏠 Home
        </a>
        <a className="nav-link" href="/calculator">
          📊 Calculator
        </a>

        <button className="btn btn-sm btn-outline-secondary mt-2 w-100"
                onClick={() => setIsDark(!isDark)}>
          {isDark ? "☀️ Hell" : "🌙 Dunkel"} Mode
        </button>
      </div>
    </nav>
  );
}

💡 Tipps

  • CSS Module-Dateien heißen genau wie die .tsx-Datei, mit angehängtem .module.css (z.B. NavMenu.module.css).
  • :global ist das React-Äquivalent zu ::deep aus CSS Modules. Syntax: :global .meine-klasse { ... }.
  • Verwende data-theme statt data-bs-theme – React hat kein Bootstrap-Naming.
  • CSS-Variablen im :root-Selektor definieren, im Dark Mode mittels [data-theme="dark"] überschreiben.
  • Responsive Breakpoints immer mobil zuerst denken: @media (min-width: 641px) für Desktop-Regeln.

✅ Akzeptanzkriterien

  • Jede Komponente hat eine eigene CSS Module-Datei
  • :global wird an mindestens einer Stelle verwendet
  • Dark/Light Mode kann per Button umgeschaltet werden
  • Die Präferenz bleibt nach Seitenneuladung erhalten (localStorage)
  • Sidebar ist unter 640px mobil nutzbar (Hamburger + Overlay)
  • CSS-Variablen für konsistente Farben im Dark Mode

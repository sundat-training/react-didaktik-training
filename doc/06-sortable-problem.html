06

Sortierbare Tabelle: Klickbare Spalten + Filter

🎯 Ziel dieser Einführung

Du erweiterst die Tabelle aus Aufgabe 05: Spaltenköpfe sind klickbar (Sortierung), und eine Button-Leiste filtert nach Buy/Sell/Alle – alles clientseitig, kein Server-Roundtrip.

🔍 Das Beispiel zum Ausprobieren

Klicke auf einen Spaltenkopf zum Sortieren. Nutze die Filter-Buttons, um nur Buy oder Sell anzuzeigen.

Typ Ort Preis Volumen
📖 Was passiert hier?

Klick auf Spaltenkopf → Sortierung. Klick auf Filter → nur passende Zeilen:

// Sortierung: Klick auf th
th.addEventListener('click', function() {
    var col = this.dataset.col;
    // toggle direction oder neue Spalte
    if (sortCol === col) sortAsc = !sortAsc;
    else { sortCol = col; sortAsc = true; }
    render();
});

// Filter: Buttons togglen active
filterBar.addEventListener('click', function(e) {
    if (e.target.dataset.filter) {
        activeFilter = e.target.dataset.filter;
        render();
    }
});

In React: Array.sort() / Array.filter() statt DOM-Manipulation. State-Variablen für sortKey, sortAsc, filter.

🧩 Übertragung nach React
  • Sort-Status: sortKey: keyof Order + sortAsc: boolean
  • Filter-Status: filter: 'all' | 'buy' | 'sell'
  • onClick auf <th> zum Sortieren
  • Sortier-Indikator: ▲ / ▼ per Conditional Rendering
  • Chain: orders.filter(...).sort(...)

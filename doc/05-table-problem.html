05

Einfache Tabelle: Daten aus einer Liste anzeigen

🎯 Ziel dieser Einführung

Du stellst eine Liste von Datenobjekten (Orders) in einer HTML-Tabelle dar – mit farblicher Kennzeichnung von Buy/Sell und einer Fußzeile mit Gesamtanzahl.

🔍 Das Beispiel zum Ausprobieren

Eine statische Tabelle mit Orders: Buy-Zeilen sind grün, Sell-Zeilen rot. Unten steht die Gesamtanzahl.

Typ Ort Preis Volumen Gesamt
📖 HTML-Struktur

Eine einfache HTML-Tabelle hat drei Teile:

<table>
    <thead>  <!-- Spaltenköpfe -->
        <tr>
            <th>Typ</th>
            <th>Preis</th>
            ...
        </tr>
    </thead>
    <tbody>  <!-- Datenzeilen — wird in React per map() befüllt -->
    </tbody>
    <tfoot>  <!-- Fußzeile mit Summen / Zählern -->
    </tfoot>
</table>

Problem: In React baust du das <tbody> nicht manuell zusammen, sondern iterierst mit map() über ein Props-Array orders. Das <tfoot> zeigt dann orders.length.

🧩 Übertragung nach React
  • Datenmodell: Order Interface mit price, volume, isBuyOrder, location
  • Komponente: OrderTable mit interface Props und orders: Order[]
  • Farben: CSS-Klasse per className Ternary setzen
  • Empty-State: {orders.length === 0 && <p>...</p>}

💡 Bevor du loslegst

  • Überlege: Wie trennst du Daten (Model) von Darstellung (View)?
  • Wie würdest du Buy/Sell farblich unterscheiden, ohne JS?
